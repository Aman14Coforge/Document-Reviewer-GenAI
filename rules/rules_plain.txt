GDP Compliance Rules for Good Documentation Practice (GDP)

Rule GDP-01 (Header, High Severity, Semantic):
Document Title on First Page Matches File Name.
Verify whether the document title on the first page matches the uploaded file name.
Recommendation: Ensure the first page title matches the file name.

Rule GDP-02 (Header, Medium Severity, Semantic):
Author Name and Role Are Mentioned.
Check whether both author name and author role are mentioned in the document.
Recommendation: Add author name and role information.

Rule GDP-03 (Revision History, Medium Severity, Deterministic):
Dates in Revision History Are Valid.
Verify that all dates in the revision history are valid calendar dates.
Recommendation: Correct invalid revision dates.

Rule GDP-04 (Revision History, High Severity, Deterministic):
Version Number Present in Title and Revision History.
Verify that version number is present in both title section and revision history section.
Recommendation: Add version information consistently.

Rule GDP-05 (Revision History, Medium Severity, Deterministic):
Revision Section Is Present.
Verify that a revision history section exists in the document.
Recommendation: Add a revision history section.

Rule GDP-06 (Approval, High Severity, Deterministic):
Signature Blocks Are Present.
Verify that approval or signature blocks exist in the document.
Recommendation: Add approval signature blocks.

Rule GDP-07 (Approval, Medium Severity, Deterministic):
Dates Present Near Signatures.
Verify that dates are present near signature blocks.
Recommendation: Add signature dates.

Rule GDP-08 (Formatting, Low Severity, Semantic):
Basic Font and Spacing Consistency.
Evaluate whether font usage and spacing appear reasonably consistent throughout the document.
Recommendation: Standardize fonts and spacing.

Rule GDP-09 (Language, Medium Severity, Semantic):
Language Errors Identified.
Identify grammar, spelling, or language quality issues.
Recommendation: Review language and grammar.

Rule GDP-10 (Structure, High Severity, Semantic):
Required Sections Present.
Verify that mandatory sections such as Objective, Scope, Responsibilities, Procedure, and References are present.
Recommendation: Add missing mandatory sections.

Rule GDP-11 (Footer, Medium Severity, Deterministic):
Page Numbers Present and Sequential.
Verify that page numbers are present and sequential throughout the document.
Recommendation: Correct page numbering.

Rule GDP-12 (Language, Low Severity, Semantic):
Readability Within Acceptable Range.
Assess whether the document readability is acceptable for its intended audience.
Recommendation: Improve readability.

Rule GDP-13 (Footer, High Severity, Deterministic):
Footer Contains Doc ID, Page Number, Confidentiality.
Verify that footer contains document ID, page number, and confidentiality statement.
Recommendation: Add required footer information.
